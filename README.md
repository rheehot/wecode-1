git add . - 깃에 전체 파일 추가
git commit -m"" - 해당 repository의 커밋 기록을 남겨준다.
git push origin main - main 브랜치를 remote로 올려준다.
git branch - 브랜치 확인
git branch feature/branchname - branchname이라는 브랜치 생성
git checkout feature/branchname - branchname이라는 브랜치로 이동
git remote add origin repository-address - remote와 local 연동
